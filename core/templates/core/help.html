Welcome to the help section!

Here you can find answers to all your questions regarding this site!

What is this site?

A little demo site to try out Django Userena (Django app that adds user registration capability). I have customized it a bit and checked how it works basically.

Which other libs are being used?

djangorestframework, django-breadcrumbs and django-widget-tweaks.

The UI is powered by jQuery and Twitter Bootstrap 3.

The icons are from Font Awesome.

How is the registration page built?

The input fields are dynamically validated with jQuery. Dynamically checking if a username is already taken is made with jQuery.ajax, which query the JSON API at http://127.0.0.1:8000/api/username_available/[username]/

Where can I find the source code?

The project is licensed under GPL version 3 or later. The code repository can be found on Github at MaTachi/django-user-registration-demo.

Which tools did you use?

Vim, Firefox and Terminator on Linux (more specific the Linux distribution Manjaro).
